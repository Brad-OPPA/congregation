# 브리프 요약 — 260507 회중의 필요

## [1] 1줄 주제
야외봉사 보조 앱(오늘의 봉사) 사용 시 개인정보 보호법을 지키며 신중함을 발휘하는 방법

## [2] 핵심 항목 (5~7개)
1. 개인정보 보호법이 보호하는 민감 정보가 무엇인지 (이름·주소·연락처·종교 성향·가족관계·관심 표시 등)
2. 야외봉사 중 앱·종이 카드·구역 도구 사용 시 조심할 점
3. 성서 원칙 — 법 준수 (로마 13:1) + 사랑·신중 (잠언 22:3) + 황금률 (마태 7:12)
4. 직접 할 수 있는 것 (앱 안에서 비공개 메모로 공식 안내·재방문 약속 정리)
5. 직접 할 수 없는 것 (앱·메모·SNS·단톡방에 다른 분의 개인정보 무단 공유, 다른 종파·정치 성향 추정 기록)
6. 격려 — 신중함은 회중의 평판을 지키고 형제 사랑을 드러낸다
7. 받는 축복 — 사람들의 신뢰, 회중의 영적 안정, 진리에 대한 좋은 증거

## [3] 강조점 (격려·축복)
- 신중함은 사랑의 표현이다 (필리피 2:4 — 자기 일뿐 아니라 다른 사람의 일도 살피라)
- 카이사르의 것은 카이사르에게 (마태 22:21)
- 회중 평판·진리에 대한 증거가 보호된다

## [4] 분량
- **10분** (브리프 명시 — 15분 아님)
- 600초 ± 30초 목표
- 도입 약 1분 30초 + 요점1 약 2분 + 요점2 약 2분 30초 + 요점3 약 2분 30초 + 결론 약 1분 30초

## [5] 전체 흐름 다이어그램
```
도입 (앱이 손에 든 작은 노트가 됐다는 현실 제시)
  → 요점 1: 성서 원칙 — 법은 준수, 동기는 사랑·신중
       성구: 로마 13:1, 잠언 22:3
       시간 마커: 1'30"
  → 요점 2: 무엇이 민감한 정보인가 — 우리가 적어두는 것의 무게
       시간 마커: 4'
  → 요점 3: 할 수 있는 것 / 할 수 없는 것
       시간 마커: 6'30"
  → 결론 + 격려 + 축복
       성구: 마태 7:12
       시간 마커: 8'30" + 종료 9'30"
```

## [6] 7개 서브에이전트별 처리할 부분 분리

### wol-researcher
- 키워드: "개인정보", "신중", "정직", "법 준수"
- 검색: wol.jw.org → 「파」 (2018-2026), 「깨」, jw.org/ko 회중관리서
- 특히 "오늘의 봉사" 앱 사용 안내 (공식 회중 봉사 안내) wol 페이지
- 종교적 삽화 — 이번 주제는 세속적이라 wol 종교 삽화 거의 불필요

### scripture-deep
- 핵심 후보 성구 (신세계역 verbatim):
  1. **로마서 13:1** — 위에 있는 권위에 복종
  2. **잠언 22:3** — 슬기로운 자는 재앙을 보고 피하지만 어리석은 자는 그대로 가다 벌을 받는다
  3. **마태 7:12** — 황금률 (남에게 받고 싶은 대로 남에게 하라)
  4. **필리피 2:4** — 자기 일뿐 아니라 다른 사람의 일도 살피라
  5. (보조) 마태 22:21 — 카이사르의 것은 카이사르에게

### publication-cross-ref
- 「파」: 정직·법 준수·이웃 사랑 관련 최근 5년 기사
- 「예-1」(여호와의 사랑 안에 머무십시오) 12장 — 정직·정부 권위
- 「조직」(하느님의 뜻을 행하는 조직) — 야외봉사 기록·회중 자료
- "오늘의 봉사" 앱 공식 안내 (jw.org 봉사자 도구)

### illustration-finder
- 도입용 일상 비유: "앱은 작은 노트와 같다 — 그러나 노트보다 멀리 퍼질 수 있다"
- 또는 "지갑에 든 다른 사람의 명함" 비유

### qa-designer
- 본 회는 응답 안 받음 (10분 연설형) — qa 블록은 결론 호소 단 1개로 단순화

### application-builder
- 4축 (가정·봉사·회중·개인) — 핵심은 봉사·개인
- "할 수 있는 것":
  - 앱에서 자기 구역의 공식 재방문 약속 (이름 이니셜 + 시간) 정도만 메모
  - 비밀번호·잠금 화면 설정
  - 앱 사용 후 즉시 닫기
- "할 수 없는 것":
  - 만난 분의 풀 이름·주소·전화번호·종교 성향을 메모창에 무단 입력
  - 메모를 단톡방·이메일로 공유
  - 다른 봉사자의 영적 약점·관심 표시를 평가 기록

### experience-collector
- 공식 출판물 게재 경험담 — "정직했더니 신뢰를 얻었다" 류 (파수대 정직 시리즈)
- 또는 "신중함이 형제를 보호했다" 류

⚠ 할루시네이션 금지 — 출판물 인용은 검증 못 한 것은 일반론으로 대체.
