# 적용 카드 3 · 축 C (회중) · 13

- **요점**: 3 (계속 왕국을 추구하라 — 밀고 나아가기)
- **축**: C (회중 — 집회 참석)
- **한 줄 적용문**: 이번 주 집회 한 번을 "빠지지 않기"가 아니라 "밀고 나아가 참석하기"로 의식적으로 재정의하고, 무엇을 밀쳤는지 집회 후 한 줄로 기록하자.
- **성구 근거**: 마태 11:12
- **예상 청중 반응·질문**:
  - (증인) "참석이 관성일 때와 적극적 쟁취일 때 느낌이 다르다."
  - 예상 질문: "집회 참석이 이미 당연한데 굳이 의식화해야 하나?" → 답: 골자의 '밀고 나아가다'는 당연함을 다시 적극성으로 재점화하는 언어.
