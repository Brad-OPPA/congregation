> ※ 위 표 누계가 30분을 넘는 것은 사회자 mic on/off 시간을 별도 분리해 표기한 것이며, **강연자 본문 합계는 정확히 1800s** = 240+180+480+720+180(결론은 420 대신 180으로 압축이 아니라, 위 표는 사회자 멘트 60+60 = 120s 를 분리 누계한 것). **재정리**:
>
> - 사회자 60s + 강연자 1800s + 사회자 60s = 총 무대 1920s
> - 강연자 1800s 분배: 서론 240 + 요점1 180 + 요점2 480 + 요점3 720 + 요점4 180 ❌ → 골자 합 = 3+8+12+7 = **30분 = 1800s 정확**
> - 따라서 강연자 1800s = 서론 240 + 요점1 180 + 요점2 480 + 요점3 720 + 결론 180. 그런데 골자 결론 = 7분(420s).
> - **계산 오류 수정**: 골자는 서론 시간 별도 표기 없이 4파트 = 3+8+12+7 = 30분. 즉 요점1 = 서론+미래를 어떻게 알 수 있는가 = 3분(180s).
>
> **확정 시간 분배 (골자 충실)**:
>
> | 강연자 구간 | 시간 |
