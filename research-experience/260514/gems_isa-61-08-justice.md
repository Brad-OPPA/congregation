# 사 61:8 — 여호와는 공의를 사랑하심 / 정직·도덕 결단 (gems_)

> 조사일: 2026-05-01
> 호출자: spiritual-gems-planner (260514, 영적 보물찾기 사회용 비상 보강)
> 용도: 공식 질문 2 (영적 보물 자율 나누기) 사회자 대비 답변 / 직장·도덕 적용
> 관련 성구: 이사야 61:8 / 미가 6:8 / 잠언 11:1 / 시편 11:7
> 관련 원칙: 여호와의 공의 사랑을 본받아 정직·결단으로 빛 비추기

## 0. 한줄 주제

사 61:8 의 "여호와는 공의를 사랑하시며 …불의를 미워하신다" — 이 본질적 특성을 본받아 직장·일상에서 정직·결단으로 빛을 비추는 형제자매들의 사례.

## 1. 후보 상세

### 후보 1 · [직장·결단] · 도널드 (Donald) — 봉사 시간 위해 직장 변경 (남아프리카)
- **출처**: 「파수대」 연구용 2024년 3월호 20-25면, 연구 기사 12 "어둠을 피하고 빛에 머무르라" 18항
- **URL**: https://www.jw.org/en/library/magazines/watchtower-study-march-2024/Avoid-the-Darkness-Remain-in-the-Light/
- **주인공**: 도널드 형제 (출판물 공개 이름) · 남아프리카
- **배경**: 직장에서 봉사 시간 확보 어려움.
- **전개** (출판물 verbatim):
  - "전파를 더 할 수 있는 일을 달라고 기도했습니다."
  - 적절한 직장 발견 → 부부가 함께 전 시간 봉사 시작.
- **교훈·원칙**: 사 61:8 의 "여호와는 공의를 사랑" — 우선순위를 영적 가치(왕국 우선)로 재배치하는 결단이 공의를 사랑하는 표현. 어둠(세상 가치 우선) 을 피하고 빛(왕국 우선) 에 머무름.
- **연결 성구**: 이사야 61:8 / 마 6:33 / 잠 3:9
- **적합도**: ★★★★★
- **주의사항**:
  - 직장 변경 일반화 금지 — 출판물 표현 그대로.
  - 결단의 결과 (전 시간 봉사) 는 명시.
- **사용 추천 위치**: Q2 — 직장 결단 적용.

### 후보 2 · [영성·소셜미디어] · 안젤라 (Angela) — 부적절한 SNS 교제 끊기
- **출처**: 「파수대」 연구용 2024년 3월호 20-25면, 동일 기사 11항
- **URL**: https://www.jw.org/en/library/magazines/watchtower-study-march-2024/Avoid-the-Darkness-Remain-in-the-Light/
- **주인공**: 안젤라 자매 (출판물 공개 이름) · 아시아
- **배경**: 성서 원칙을 존중하지 않는 사람들과 SNS에서 교제.
- **전개** (출판물 verbatim):
  - "그것은 함정이 될 수 있어서, 감각을 둔하게 합니다." (11항)
  - 장로의 도움으로 영적 콘텐츠로 재집중.
- **교훈·원칙**: 사 61:8 의 "여호와는 …불의를 미워하신다" — 미묘한 영적 위험에서 멀어지는 결단이 공의 사랑의 표현. 보이지 않는 어둠(SNS) 도 빛에서 멀어지게 함.
- **연결 성구**: 이사야 61:8 / 잠 13:20 / 고전 15:33
- **적합도**: ★★★★★ (SNS 적용 신선)
- **주의사항**:
  - SNS 사용 자체를 정죄하지 않게 — 부적절한 교제로 한정.
  - 청년·청소년 청중에게 즉시 공감.
- **사용 추천 위치**: Q2 — 영성·SNS 적용.

### 후보 3 · [회복·정직] · 아서 (Arthur) — 죄 고백·회복 (사 61:1, 61:8 평행)
- **출처**: 「파수대」 연구용 2025년 5월호, 연구 기사 20 "여호와께 위로를 구하라"
- **URL**: https://www.jw.org/en/library/magazines/watchtower-study-may-2025/Look-to-Jehovah-for-Comfort/
- **주인공**: 아서 형제 (출판물 공개 이름)
- **배경**: 약 1년간 음란물과 씨름.
- **전개**:
  - 아내·장로에게 고백 (정직).
  - 장로 격려: "여호와께서 너를 거절하지 않으셨다."
  - 회복 → 보조 봉사자·개척자.
- **교훈·원칙**: 사 61:8 의 "공의를 사랑" — 죄를 인정하고 고백하는 정직이 공의에 대한 사랑의 표현. 사 61:1 의 자유 선포와 평행.
- **연결 성구**: 이사야 61:8 / 시 32:5 / 요1 1:9
- **적합도**: ★★★★☆
- **주의사항**:
  - 청소년 많은 집회는 주제 민감 — 정직·고백 측면만 일반화.
  - 직접 단어 사용 금지 — 출판물도 "씨름" 으로 표현.
- **사용 추천 위치**: Q2 — 정직·고백 적용.

### 후보 4 · [공의·여호와의 본질] · 사 61:8 vs 미 6:8 평행 풀이
- **출처**: 「통찰」 / 「파」 24.7 30-31면 평행 풀이
- **URL**: https://wol.jw.org/ko/wol/pc/r8/lp-ko/202026162/1/0
- **주인공**: 풀이 자체 (집단 적용)
- **배경**: 사 61:8 ("여호와는 공의를 사랑하시며") + 미 6:8 ("공의를 행하며 친절을 사랑하며 …하느님과 함께 겸손히 걸어가는 것") 의 평행.
- **전개**:
  - 두 성구가 여호와의 핵심 특성과 인간의 응답을 짝지움.
  - 사 61:8 = 여호와의 본질 / 미 6:8 = 인간의 응답.
  - 공의는 단순한 규칙이 아닌 사랑의 표현.
- **교훈·원칙**: 사 61:8 영적 보물 — 여호와의 공의 사랑은 응답을 요청. 직장·가정·회중에서 공의를 행하는 것이 그 응답.
- **연결 성구**: 이사야 61:8 / 미 6:8 / 시 11:7 / 마 7:12
- **적합도**: ★★★★★ (성구 평행 + 영적 보물 깊이)
- **주의사항**:
  - 풀이 자체 — 표준 교리, 검증 완료.
- **사용 추천 위치**: Q2 사회자 답변 — 사 61:8 영적 보물 핵심.

### 후보 5 · [환경·정의] · 사 61:8 의 "강도질·불의 미워" → 회사 부패 거부 일반 원칙
- **출처**: 「파수대」 일반 노선 — 정직 주제 (구체 호수 verbatim 미확인 ⚠️)
- **URL**: 「파」 정직 시리즈 다수 (verbatim 검증 필요 — 사회자가 직접 확인 권장)
- **주인공**: (집단 적용 — 정직으로 알려진 그리스도인)
- **배경**: 회사 부정·뒷돈 거부.
- **전개**:
  - 사 61:8 의 "내가 강도질을 …미워한다" 는 모든 형태의 부정 거부의 근거.
  - 직장 부정 거부로 손해 보는 듯하지만, 여호와의 공의 사랑을 본받음.
- **교훈·원칙**: 사 61:8 의 강도질 거부 — 작은 부정도 여호와께서 미워하심 → 그리스도인의 직장 정직.
- **연결 성구**: 이사야 61:8 / 잠 11:1 / 히 13:18
- **적합도**: ★★★☆☆ (구체 인물 verbatim 미확보)
- **주의사항**:
  - **⚠️ 사회자 사용 전 verbatim 출처 보강 필수** — 추상적 적용으로만 사용 권장.
- **사용 추천 위치**: Q2 — 직장 정직 일반 적용 보조.
