# 🎙️ 회중의 필요 v6 연설형 패턴 (2026-05-07 원준님 확정)

> ver4 표준 (`local-needs-ver4-standard.md`) **위에 누적** 되는 연설 전달 패턴.
> 회중의 필요 는 글 읽기가 아니라 **연설** 이라 — 시각 구조 (### 헤더, bullet) 보다 **음성 흐름 (서술형 prose)** 우선.

## P1. 5단 성구 패턴 (도입 한 번만)

도입의 첫 성구는 **5단 형식 갖춤**:

```
[1. 소개 문장]: "성서는 [주제]에 대해 이렇게 말합니다."
[2. 낭독 안내]: "함께 [참조] 보시겠습니다."
[3. (낭독)]:   stage direction (script 안에 표시, 음성 X)
[4. 성구 verbatim 본문]
[5. 핵심 어구 추출 + 관련 설명]: "핵심은 [Y]"...."[/Y]는 부분입니다. ..."
```

예시 (도입):
> 성서는 슬기로운 자에 대해 이렇게 말합니다. **함께 잠언 22:3 보시겠습니다.** (낭독)
> "[verse]"
> 핵심은 [Y]"위험을 보면 몸을 숨긴다"[/Y]는 부분입니다. ...

## P2. 서술형 prose (요점 1·2·3·결론 모든 후속 성구)

도입 이후 성구는 **자연스러운 산문 통합**:

```
[1. 평문 도입]:    "ㅁㅁ장 ㅁ절을 보면 이렇게 말씀합니다 —"
[2. 성구 verbatim] (INDENT 블록 유지)
[3. 핵심 + 설명]:  "여기서 핵심은 [Y]"..."[/Y]는 부분입니다. ..."
```

**금지** — 도입 이후 성구에서:
- "함께 [참조] 보시겠습니다" 형식 갖춤 X
- "(낭독)" stage direction X

## P3. 한 요점 = 한 성구 낭독

**규칙**: 한 요점에 1 성구만 INDENT 블록으로 낭독. 나머지 성구는 짧은 산문 인용으로 통합.

예시 (요점 1, 잠 14:15 + 로마 13:1):
- **낭독**: 잠 14:15 (INDENT 블록 — 핵심 주제 직결)
- **참조 only**: "사도 바울도 로마서 13장 1절에서 [Y]'위에 있는 권위에 복종하라'[/Y]고 권했습니다" (산문 안 짧은 인용)

## P4. WOL 신세계역 **연구용 성서 (nwtsty)** verbatim (절대)

회중에서 사용하는 모든 성경 인용은 **신세계역 성서 연구용 판** 만 사용 (원준님 확정 2026-05-07).

- WOL URL 패턴: `https://wol.jw.org/ko/wol/b/r8/lp-ko/nwtsty/{book_num}/{chap}` — `nwtsty` 가 연구용 성서
- 검증 도구: `_automation/scrape_wt.py::fetch_nwt_verse()` (이미 nwtsty 사용 중) 또는 fact-checker 에이전트
- 사용자 검수 의존 0 — "검증 권고"·"검토 바람" 부록 메모로 우회 X

### 연구용 성서 책 이름 (일반판과 다른 항목 주의)

| 책 | 일반 NWT 2014 | 연구용 성서 (정본) |
|---|---|---|
| 시편·잠언·전도 등 | 잠언 | 잠언 ✓ |
| 마태 | 마태복음 | 마태복음 ✓ |
| 로마 | 로마서 | 로마서 ✓ |
| **빌립보** | 필리피 | **빌립보서** |
| **데살로니가** | 테살로니카 | **데살로니가전서·후서** |
| **유다서** | 유다 | **유다서** |

## P5. 묶음 처리 — 3 영역 음성 cue

요점 2 같이 "할 수 없는 것 N가지" 가 5개 이상이면 **3 묶음** 으로 정리. ### 시각 헤더 X — 음성 cue 로:

```
이제 가장 중요한 부분, [Y]슬기롭게 피해야 할 것들[/Y]입니다.
외우기 쉽도록 [Y]세 영역[/Y]으로 정리해 보겠습니다.

[Y]첫째 영역은 'XXX'[/Y]입니다. ... (단락)

[Y]둘째 영역은 'YYY'[/Y]입니다. ... (단락)

[Y]셋째 영역은 'ZZZ'[/Y]입니다. ... (단락)

이렇게 [Y]세 영역에서 분별할 때 우리는 슬기롭게 행동할 수 있습니다.[/Y]
```

청중에게는 "**첫째 영역은 ...입니다**" 같은 음성 cue 로 들려 — 시각 헤더 없이도 구조 파악 가능.

## P6. 어휘 선택

| 사용 ✓ | 회피 ✗ | 이유 |
|---|---|---|
| **출처** (앱의 출처) | 정체 | "정체" 는 신비감/숨김 암시 — 사제 앱 출처 가리키는 데 부적합 |
| **순진한 자** (잠 14:15) | 경험 없는 자 | NWT 2014 verbatim |
| **치른다** (잠 22:3) | 친다 | NWT 2014 verbatim |
| **조심스러우면서도** (마태 10:16) | 슬기롭고 | NWT 2014 verbatim. 본문 "슬기로움" 테마와는 commentary 다리로 연결 ("조심스러움 = 슬기로운 신중함") |
| **분별력이 너를 보호하여** (잠 2:11) | 보호해 줄 것이다 | NWT 2014 verbatim |

## P7. 도입 — 익명의 관찰자 framing

봉사 중 형제가 가정 문 앞에 서 있는 장면 묘사 시:
- ✗ "옆에서 본 집주인" (집주인이 안에 있는데 "옆"이라 어색)
- ✓ "**형제 자신은 모르지만, 누군가 그 모습을 지켜보고 있을 수 있습니다**" (익명의 관찰자, 형제 무자각 강조)

긴장 구조: 형제의 *순수한 의도* ↔ 관찰자의 *다른 인상*.

## P8. 도입 — 구체적 디테일 명시

추상적 "관련 내용" → **구체적 예시** :
- ✗ "관련 내용을 입력합니다"
- ✓ "[Y]재방문 약속, 집주인 관련 메모, 부재 표시[/Y] 같은 봉사에 도움이 될 만한 내용입니다"

구체적 예시는 talk 후반 "피해야 할 것" 과 직결되어 일관성 ↑.

## P9. JW 표준 — 구역 / 가정

봉사 위치 묘사 시 **JW 한국어 표준 용어**:
- ✓ "**구역 내 어느 집 문 앞**" (JW 야외봉사 맥락 명확)
- ✗ "한 가정 앞" (모호)
- ✗ "가정 문 앞" (어색한 합성어)
- ✗ "현관 앞" (일상 표현, JW 맥락 약함)

## P10. "우리가 전하는 진리에 대한 인상"

도구 사용이 줄 수 있는 인상에 대해 말할 때:
- ✗ "진리에 대한 인상" (진리가 무엇의 진리인지 모호)
- ✓ "[Y]우리가 전하는 진리에 대한 인상[/Y]" (야외봉사 맥락 명확)

## P11-bis. 경계 메시지 — 상황 조건 명시

"이런 행동을 하지 마십시오" 식 경계 메시지 앞에 **언제 적용되는지 상황 조건** 을 한 절로 박으면, 청중이 즉시 자기 일상 속 그 순간을 떠올림.

| ✗ 무조건적 경계 | ✓ 조건 박은 경계 |
|---|---|
| "이 앱을 공식 앱처럼 말씀하지 마십시오" | "[Y]누군가가 이 앱에 대해서 묻게 된다면[/Y], 이 앱을 공식 앱처럼 말씀하지 마십시오" |
| "외부에 노출하지 마십시오" | "단톡방·이메일·SNS 같은 [Y]회중 밖 채널을 쓰게 될 때[/Y], 그 메모를 옮기지 마십시오" (가능 시) |

장점: (1) 조건이 청중의 머릿속에 구체적 장면을 띄움, (2) 무조건적 금지보다 부드러움, (3) "그런 상황이 오기 전에 미리 정해 두라" 의 슬기로움 주제와 연결.

## P12. 결론 — 반복 어구 회피

결론 첫 문장과 마지막 문장에 같은 표현 (예: "풍성한 축복으로 갚으십니다") 반복 X.
- 첫 문장: "이렇게 슬기롭게 행할 때, 사람들은..." (전환만)
- 마지막 문장: "여호와께서는 슬기롭게 행하는 그 종을 기쁘게 여기시며 풍성한 축복으로 갚아 주실 것입니다" (메인 약속)

## P13. 사전 검증 게이트 (CLAUDE.md 의무)
