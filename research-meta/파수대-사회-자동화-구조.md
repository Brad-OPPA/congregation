## 4. urllib timeout 회피 — requests shim (Mac 환경)

macOS Python 3.14 의 urllib 이 wol.jw.org 에 대해 timeout 발생. 해결:

```python
import urllib.request, urllib.error
import requests

_session = requests.Session()
_session.headers.update({'User-Agent': 'Mozilla/5.0 ...'})

class _FakeResp:
    def __init__(self, content, headers):
        self._content = content; self._headers = headers
    def read(self):  return self._content
    def info(self):  return self._headers
    def __enter__(self): return self
    def __exit__(self, *a): return False

def _shim_urlopen(req, *a, timeout=30, **k):
    url = req.full_url if hasattr(req,'full_url') else req
    headers = dict(req.headers) if hasattr(req,'headers') else {}
    r = _session.get(url, headers=headers, timeout=timeout, allow_redirects=True)
    if r.status_code >= 400:
        raise urllib.error.HTTPError(url, r.status_code, r.reason, r.headers, None)
    return _FakeResp(r.content, r.headers)

urllib.request.urlopen = _shim_urlopen
```

`scrape_wt.py` import **이전에** shim 설치 필수.

## 5. add_cue() 깊이 주입 패턴

베이스 spec 의 각 block 의 `sequence` 리스트에 `host_cue` 항목 추가:

```python
def add_cue(numbers, runs):
    for b in spec['blocks']:
        if b['numbers'] == numbers:
            b['sequence'].append({'type': 'host_cue', 'runs': runs})
            return
```

`runs` 는 (text, flags) 튜플 리스트. flags:
- `''` 평문
- `'b'` 볼드 (출판물 인용 키워드)
- `'i'` 이탤릭
- `'y'` 노랑 하이라이트 (사회자 핵심 대사)
- `'r'` 빨강 (성구 verbatim — 신세계역)
- `'B'` 파랑 #0070C0 (보조 강조)

블록 numbers 가 합쳐진 경우 ('8-9', '14-15') 도 정확 매칭. WOL 원문대로 ㄱ·ㄴ·합산 번호 보존.
