## 1. 호출 체인 (확정)

```
[원준님 입력 1회] /publictalk 179 (또는 대화형 번호 입력)
   ↓
[A] public-talk-builder = 기획자
    - 골자 PDF Read (S-34_KO_NNN.docx 또는 PB_NNN-KO*.pdf)
    - 30분 설계도 + 6개 에이전트 지시서 작성
    - 저장: research-public-talk/{NNN}_{YYMMDD}.md
   ↓
[B] 6개 에이전트 병렬 (한 메시지 동시 호출)
    - scripture-deep (성구 ≥ 10 후보)
    - illustration-finder (예화 ≥ 30 후보 + 시각자료 ≥ 8 후보)
    - experience-collector (경험담 ≥ 10 후보)
    - application-builder (적용 카드 ≥ 12 후보)
    - publication-cross-ref (출판물 횡단 ≥ 10 후보)
    - qa-designer (mode 2 — 후크 5종 × A/B/C 3안 + 수사적 질문 ≥ 15 + 인터랙션 ≥ 5)
   ↓
[B-3] public-talk-builder 1차 재검수 — R 룰 grep
    - R1·R3·R4·R5·R6·R8·R-J5 후보 풍부도 검증
    - 미달 시 해당 보조 재호출
   ↓
[C] public-talk-script — 30분 서술형 원고 작성
    - 표어 성구 verbatim (NWT)
    - 서론 5 흐름 (후크 + 문제제기 + 연결 다리 + 표어 성구 + 강연 예고)
    - 요점 1·2·3 = 흥미 유발 → 성구 낭독 → 설명 → 예화/경험담 → 적용 → 전환
    - 결론 = 요점 복습 + "오늘 배운 3가지" 명시 + 행동 촉구 + 감정 몰입 장면
    - 저장: research-public-talk/{NNN}_{슬러그}_원고_ver{N}_{YYMMDD}.md
   ↓
[C-2] public-talk-builder 2차 재검수 = 기획자 최종 QA
    - R1~R20 + R-Conv + R-J1~J5 정량 grep
    - HIGH 위반 시 script 재호출
   ↓
[D] 스펙 파일 생성 — _automation/content_publictalk_{NNN}.py
   ↓
[E] 빌드 — python3 _automation/build_publictalk.py
    - validators.validate_md_text() 자동 차단 (§2 + §2-bis + §2-ter)
    - docx2pdf (Word) → 실패 시 soffice --headless 자동 fallback
    - 출력: ~/Dropbox/.../S01.공개강연/김원준 공개강연/{NNN}_{슬러그}/
   ↓
[F] 4종 게이트 자동 호출 (한 메시지 병렬)
    - fact-checker → research-factcheck/
    - jw-style-checker (NG list + 의심 어휘 grep) → research-style/
    - timing-auditor (1800±120초) → research-timing/
    - quality-monotonic-checker (9축) → research-quality/
   ↓
[F-2] 재감수 게이트 (재빌드 시 한 번 더) — 최대 _re2_
   ↓
[F-3] ⑨ 결과 처리 — 자동 분기
    - 4종 PASS → 단계 G
    - timing 단독 FAIL + quality PASS → 단계 G (quality > timing)
    - fact / style / quality FAIL → 해당 영역 재호출 → 단계 D~F 재실행
    - 자동 재작성 5회 한도 (ver1 → ver5) → 초과 시 사용자 BLOCKING
   ↓
[G] 사용자 검수 1회 — docx/PDF 직접 확인
   ↓
[H] git 자동 커밋 → push (Brad-OPPA/congregation)
   ↓
[완료]
```

= 사용자 개입 = **입력 1회 + 검수 1회**.

## 5. 출처·외부 자료 우선순위 (서론·예화)

```
1순위: 외부 1차 자료 — Britannica · NASA · Nat'l Geographic · Smithsonian · 박물관 공식 · 공식 전기 · 공개 연설 녹취 · 동료 심사 논문 · 신뢰 언론
2순위: JW 출판물 중 20년 이상 된 자료 (기억에서 흐려진 것)
3순위: JW 출판물 최근 10년 자료 — 되도록 회피
```

**이유**: 공개강연 청중 대부분이 전도인. 최근 10년 파수대·깨어라 그대로 쓰면 "그 기사" 인식 → 귀가 닫힘. 신선함이 첫 30초 성패.

**시각자료 별도 우선순위**:
```
1순위: wol.jw.org 기사 본문 삽화 (해당 강연 골자 PDF 명시 삽화 + 관련 파/깨/훈 본문 삽화)
2순위: jw.org 도서관 이미지
3순위: JW 방송 영상 캡처 (tv.jw.org)
4순위: 외부 — 위키커먼즈 · 박물관 공개 고해상도 · 공개 도메인 (상업 스톡 금지)
```

종교 내용 이미지 (성경 장면·왕국 진리·신세계 장면) 는 **wol/JW방송 의무**, 외부 불가. 외부는 세속 사실 (고고학·지형·자연·과학·일상 사물) 만.
