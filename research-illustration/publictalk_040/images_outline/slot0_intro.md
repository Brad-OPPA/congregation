# Slot 0 — 서론용 시각 자료 (요점1)

## 주제
인간 예측 실패 vs 성경 예언 적중 (서론 후크 시각화)

## 검색 시도 기록

### 1순위 (wol.jw.org 본문 삽화)
- **시도 1**: 「파수대」 2019년 9월호 연구용 “아마겟돈—좋은 소식!” 의 6패널 종합 삽화
  - URL (한국어 본문): https://wol.jw.org/ko/wol/d/r8/lp-ko/2019561
  - 삽화 이미지 URL: https://wol.jw.org/ko/wol/mp/r8/lp-ko/w19/2019/643
  - alt: “아마겟돈으로 이어지는 사건들이 있을 때 세상 사람들과 여호와의 백성이 어떤 대조적인 반응을 나타낼 것인지 보여 주는 6개의 삽화”
  - **채택**: 서론에서 “세상의 예측은 빗나가지만 여호와께서 알려 주신 일은 6단계로 명확하게 펼쳐진다” 라고 도입할 때 6패널 전경 슬라이드로 사용 가능
