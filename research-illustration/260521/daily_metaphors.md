## 요점 1 — 동정심 (여호와께서 우리 실수의 고통 가운데 함께하심)

### 비유 1-A — 우는 자녀를 안아 주는 어머니

- **장면**: 어린아이가 컵을 떨어뜨려 깨고 자기 손을 베인 채 운다. 어머니는 "왜 그랬어!" 라고 꾸짖기 전에 먼저 아이를 끌어안고 손의 상처를 살핀다.
- **본 주차 떠받침**: 사 63:9 — "그분도 괴로워하셨다 ··· 들어올리시고 운반하셨다." 잘못의 결과로 아파하는 자녀의 고통을 부모가 함께 느끼는 모습이 본 주차 메시지의 핵심.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 1-B — 환자의 통증을 함께 살피는 의사

- **장면**: 진료실에서 의사는 환자의 자기 부주의로 생긴 부상이라도 "그건 본인 잘못이잖아요" 라고 말하지 않는다. 먼저 통증의 정도를 묻고, 표정을 살피고, 손을 환부에 가까이 대 본다.
- **본 주차 떠받침**: 동정심(영어 compassion = "함께 아파한다")의 어원과 직결. 결과를 자초했어도 고통은 진짜고, 그 고통에 먼저 반응하는 태도.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 1-C (보강) — 넘어진 친구를 일으켜 주는 친구

- **장면**: 빙판길에서 친구가 미끄러져 넘어졌다. "내가 조심하라고 했지" 라는 말 대신 먼저 손을 내밀어 잡아 일으키고, 옷에 묻은 흙을 털어 준다.
- **본 주차 떠받침**: 사 63:9 의 "들어올리시고" 와 직접 매칭되는 일상 행동.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

## 요점 2 — 잘못의 결과를 막지 않으나 충실 도우심

### 비유 2-A — 운전 면허 시험에 떨어진 사람의 재도전

- **장면**: 운전 면허 시험에 떨어졌다고 해서 시험관이 "다음에는 내가 통과시켜 줄게" 라며 결과를 바꿔 주지는 않는다. 떨어진 사실은 그대로다. 그러나 가족이나 친구는 "어디서 실수했는지 같이 보자" 며 옆에서 다시 연습할 힘을 준다.
- **본 주차 떠받침**: 「파23.08」 28-29면 8-9항 — 해로운 결과는 막지 않으시지만 충실을 유지하도록 도와주신다. 결과는 결과대로, 도움은 도움대로.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 2-B — 자전거에서 처음 넘어진 아이가 다시 일어남

- **장면**: 처음 자전거를 배우는 아이가 균형을 잃고 넘어진다. 무릎이 까지고 손바닥에 자갈이 박힌다. 부모는 상처를 만들어 준 것은 아니지만, 그 상처를 닦아 주고 다시 안장에 앉도록 손을 잡아 준다. 다음 날 아이는 같은 길에서 한 번 더 도전한다.
- **본 주차 떠받침**: 잘못된 선택의 결과(까진 무릎)는 그대로 따라오지만, 충실을 유지하도록 도와주시는 도움 → 다시 일어남.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 2-C (보강) — 빚을 진 사람이 가계부를 다시 쓰기 시작함

- **장면**: 충동적으로 카드를 긁어 빚이 쌓였다. 빚 자체는 그대로 갚아 나가야 한다. 그러나 옆에 있는 가족이 "처음부터 다시 시작하자" 며 가계부 쓰는 법을 옆에서 알려 주면, 같은 실수를 반복하지 않고 충실히 살아갈 힘이 생긴다.
- **본 주차 떠받침**: 결과(빚)는 그대로지만, 충실 유지를 위한 구체적 도움이 옆에서 이어짐.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

## 요점 3 — 순응 → 귀중한 사람으로 빚으심

### 비유 3-A — 도자기 공방에서 진흙을 빚는 손

- **장면**: 도자기 공방에서 도공의 양손이 회전하는 진흙 덩이 위에 닿는다. 진흙은 너무 마르면 갈라지고, 너무 묽으면 무너진다. 적당히 부드러워야 손가락 끝의 작은 압력에도 형태를 따라간다. 도공의 시선은 결코 진흙에서 떠나지 않는다.
- **본 주차 떠받침**: 사 64:8 — "주는 토기장이." 부드러운 진흙 = 순응 / 손가락의 압력 = 인도 / 형태가 잡혀 감 = 귀중한 사람으로 빚어짐.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 3-B — 청소년의 인격이 매일 작은 선택으로 다듬어짐

- **장면**: 청소년이 매일 학교에서 부딪히는 작은 결정 — 친구가 거짓말하자고 부추길 때 거절할지, 화가 나도 한 번 호흡을 가다듬을지. 큰 사건 한 번이 아니라 작은 결정이 쌓여 인격이 다듬어진다. 일 년 뒤 돌아보면 그 사람의 모양이 바뀌어 있다.
- **본 주차 떠받침**: "매일 작은 결정에서 여호와의 인도에 순응" 적용 카드와 직결. 빚어짐은 한순간이 아니라 누적 과정.
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"

### 비유 3-C (보강) — 정원사의 손에 다듬어지는 묘목

- **장면**: 정원사가 어린 묘목의 가지를 매주 조금씩 묶어 주고, 휘어진 곳을 똑바로 세워 준다. 묘목은 처음에 가는 줄기지만, 정원사의 손길이 방향을 정해 준 대로 자라 일 년 뒤 곧고 튼튼한 나무가 된다.
- **본 주차 떠받침**: 인도에 순응 → 형태가 잡힘. 도공 비유와 결을 맞춘 또 다른 일상 비유 (서론 외부 후크 후보로도 사용 가능).
- 메타: supports_main_paragraph=true / where_to_place="예" / non_other_religion=true / fact_verified=true / axis="일상"
