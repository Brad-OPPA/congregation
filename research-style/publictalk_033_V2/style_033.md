| 항목 | 대체 | 이유 |
|---|---|---|
| A-1. "예배" | **"집회" / "연설" / "공개 강연" / "하느님께 드리는 봉사"** | JW 공식 한국어에서 모임을 "예배"로 부르지 않음. 회중 메모리 feedback_congregation_terminology.md. |

| A-4. "교회" | "회중" / "왕국회관"(건물) | JW는 "교회" 용어 미사용. |

| A-7. "주님" (예수를 지칭할 때 단독 사용) | "주 예수 그리스도" / "예수" | 여호와와의 혼동 방지. 강연 중 1회도 쓰지 않는 편이 안전. |

## B. "여호와" 첫 등장 시 부연 (비증인 청중 배려)

- **원고에서 "여호와"가 처음 나오는 문장**에 반드시 부연을 붙인다.
- 승인된 부연 템플릿 (골자·파수대 공개강연 관례):

> "하느님의 이름은 **여호와**입니다. 시편 83편 18절을 보면 '홀로 지극히 높으신 분'으로 소개됩니다."

또는 더 짧게:

> "성경이 밝히는 하느님의 이름, **여호와**는…"

- 이후 등장부터는 부연 없이 자연스럽게 "여호와"만 사용.
- 예수를 말할 때 "여호와의 아들 예수 그리스도"로 첫 등장 시 관계 명시 권장.

**점검 포인트**: 블록 A 서론 또는 블록 B 신 32:4-5 설명부에서 첫 "여호와"를 찾아 부연 유무 확인.

## C. 공식 용어 일관성

| 개념 | 공식 용어 | 지양 |
|---|---|---|
| 성경 | **"성경"** (일반) / **"신세계역"** (본 강연에서 낭독하는 역본 명시 필요 시) | "성서"(출판물 제목 포함 시는 허용) |
| 하느님 | **"하느님"** (한국어 공식) | "하나님" (한국 타 교단 표기) |
| 모임 공간 | **"왕국회관"** | "교회 건물" |
| 회중 단위 | **"회중"** | "교회" |
| 왕국 | **"하느님의 왕국" / "하늘 왕국"** | 막연한 "천국" |
| 경전 약어 | 「파수대」 → **「파」**, 「깨어라!」 → **「깨」**, 「통찰」 → **「통」** | 자의적 약어 |
| 예수 호칭 | **"예수 그리스도"** 또는 **"예수"** | "주" 단독 |
| 출판사 | **"워치타워 성서 책자 협회"** (언급 필요시만) | 다른 표기 |

**2026년 최신 특이 사항**:
- 2024년 신세계역 개정판 공식 사용 중. **성구 낭독 시 신세계역 본문 기준**(이 강연 낭독 성구 3개: 전 4:1, 사 9:6-7, 벧전 5:6-7).
- "여호와"는 2020년대 이후 공식 한국어 표기 고정. **"야훼" 표기 금지**.

## D. 정치 중립 표현 가이드 (요점 2 "맞서 싸우는 것" 블록에서 특히 주의)

### D-1. 절대 금지
- 특정 **국가명 + 부정적 수식** ("A나라의 독재", "B나라의 실패")
- 특정 **정당명**, **정치인 실명**, **현직 지도자**
- 특정 **사회 운동명 옹호/비판** (예: 페미니즘·환경운동·노동운동 개별 평가)
- 종교 지도자 **실명 비난**

### D-2. 허용되는 중립 서술 틀
- **역사적 사실로서의 기록 제시**만 허용:
  - 예: "2010년 튀니지에서 한 청년의 분신 항의가 한 지역 전체의 봉기를 촉발했지만, 십여 년이 지난 지금 그 지역이 더 정의로워졌다고 말하기는 어렵습니다." → **사실 서술 + 평가는 성경 원칙으로**.
- 일반 명사로 추상화: "어떤 나라", "한 지역", "여러 사회 운동"

### D-3. 요점 2 블록에서 사용할 **중립 표현 템플릿**

**좋은 예 (중립)**:
> "역사적으로 많은 사람들이 불공정에 맞서 거리로 나섰습니다. 그들의 의도는 이해할 수 있는 것이었습니다. 그러나 근본 원인이 바로잡히지 않은 채 제도만 바뀌면, 같은 문제가 다른 형태로 다시 나타났습니다. 성경은 이 현실을 솔직하게 인정합니다."

**나쁜 예 (편향)**:
> "○○혁명은 결국 실패했습니다. ○○주의는 인간 본성을 무시합니다." — 금지.

### D-4. 군중·시위 이미지 해설 시
- "이 사람들처럼"은 허용(이미지 속 인물 지칭).
- 이미지 속 **특정 깃발·현수막이 식별 가능한 경우 해당 이미지 사용 재고**. 중립 이미지로 교체.

### D-5. "맞서 싸움"에 대한 언어 (골자 006~011 문맥)
- **부드럽게**: "이해는 가지만 근본 해결은 되지 않습니다", "인간의 능력에는 한계가 있습니다", "성경은 그 이유를 이렇게 설명합니다"
- **피할 표현**: "그들은 어리석다", "무의미하다", "실패할 수밖에 없다"(감정적으로 단정적) → 전 1:15·렘 10:23 인용을 통해 **성경이 말하게** 하라.

## E. 청중 공감 톤 (비증인 배려)

| 위치 | 권고 |
|---|---|
| 서론 | 뉴스 제목 나열·"여러분도 느끼셨을 것입니다" → 공감으로 출발, **교리 선언으로 시작하지 않음** |
| 성구 인용 | "성경에는 이런 말이 있습니다"식 도입 → 비증인에게 성경이 낯선 경우 대비 |
| 낭독 시 | 구절 주소를 먼저 **한 번만** 또렷이 읽고(예: "전도서 4장 1절") 본문 낭독 |
| 직접 호칭 | "여러분" 중심, "우리 증인들은"은 1~2회로 제한 |
| 결론 | 초대형 행동("바로 가입하세요") 아님. **"성경을 더 살펴보시면 좋겠습니다"**로 마무리 |

## F. 시각 자료 설명 시 용어

- "시각 자료 1·2" (골자 지정) — 원고에서 `[시각 자료 1]`·`[시각 자료 2]`로 표기 가능하나, **낭독 시에는 "이 사진"·"이 그림"** 으로 자연스럽게 풀기.
- 추가 이미지 해설 시 **출처 불명 이미지는 침묵**(해설로 만회), 필요시 "한 역사적 장면"·"많은 나라에서 볼 수 있는 장면"식 중립화.

## G. 시간·수치 서술

- 연도·날짜: 한국어 공식 서식 "2010년", "2013년 7월호", "2020년대"
- 성구 표기 형식:
  - 본문 낭독 직전: "**전도서 4장 1절**" (정식)
  - 본문 괄호 인용: "(전 4:1)"
  - 원고 내 교차 참조 메모: "전 4:1" (공백 1칸 허용)
- **"BC/AD" 대신 "기원전/기원후"** (JW 공식)

## H. 최종 자동 점검 리스트 (원고 작성 후 실행)

**단순 grep 기반 (Python/ripgrep)**:
1. `예배` — 0회 (출현 위치 있으면 "집회"로 교체)
2. `하나님` — 0회 (→ "하느님")
3. `천국` — 0회 (→ "하늘 왕국")
4. `교회` — 0회
5. `주님\b` — 0회 (예수 지칭 시 "예수 그리스도"로)
6. `야훼` — 0회

**논리 점검 (사람 검토)**:
7. 첫 "여호와" 등장부에 부연 있는가?
8. 낭독 지정 성구 3개(전 4:1 / 사 9:6-7 / 벧전 5:6-7) 낭독 마커 누락 없는가?
9. 특정 국가·정당·인물 실명 0회인가?
10. 출판물 약어 표기 통일되었는가?
11. 결론이 "성경을 더 살펴보시기 바랍니다" 기조인가(가입 권유가 아니라)?
12. 총 시간 배분 30분 이내에 맞는가? (각 블록 ±15초 허용)

**금지 구문 추가 점검**:
13. "우리 종교가 맞다" / "다른 종교는 틀렸다" 유사 표현 — 0회. 성경이 말하게 하라.
14. "반드시 ○○해야 구원받는다" 식 조건형 구원 표현 — 피함 (청중 배려).

## I. 체크 항목 총합

- C(공식 용어): 5 (성경/하느님/왕국회관/약어/예수 호칭)
