# 호칭 가이드 — 공개강연 179 V2

**전제**: 청중의 15-30%는 비증인. 호칭은 "회중 내부 대화"가 아니라 "손님이 함께 있는 자리에서의 대화"로 설계한다.

## 1. "형제자매 여러분" vs "청중 여러분" 비율 권고

| 국면 | 권장 호칭 | 사용 비율(전체 30분 중) |
|------|----------|------------------------|
| 서론 (0-2분) | **"청중 여러분"** / **"여러분"** | 100% "청중"/"여러분" 톤. "형제자매" 금지 (비증인 초대 장면). |
| 요점 1 (2-17분) | "여러분" 중심, 필요 시 "형제자매 여러분" 1-2회만 | 형제자매 호칭 10%, 여러분/청중 90% |
| 요점 2 (17-26분) | "형제자매 여러분" 점진적 증가 | 형제자매 30%, 여러분 70% |
| 결론 (26-30분) | **"형제자매 여러분, 그리고 오늘 저희와 함께 자리해 주신 여러분"** 병렬 호칭 권장 | 병렬형 주력 |

**근거**:
- 서론에서 "형제자매"로 시작하면 비증인이 "나는 여기 낄 자격이 없다"는 신호를 받는다.
- 반대로 끝까지 "청중"으로만 부르면 여호와의 증인의 정체성·공동체성이 희석된다.
- 따라서 **전반부는 포용 톤, 후반부는 정체성 톤**으로 점진 이행.

## 2. "우리" 대명사 사용 가이드

### 2-1. 안전한 "우리" (비증인 포함 가능한 맥락)

- **보편적 인간 경험**: "우리 모두는 환상에 끌리기 쉽습니다." ✓
- **성서 독자 공동체**: "우리가 방금 읽은 야고보 1장은…" ✓
- **현대 시대 공유**: "우리가 사는 이 시대에는…" ✓

### 2-2. 위험한 "우리" (비증인을 배제하는 맥락)

- ❌ "우리 여호와의 증인은 매주 전도를 합니다." → 비증인은 "우리"에 포함되지 않음.
  - ✓ 교정: "여호와의 증인들은 매주 전도를 합니다." (주어 명시)
- ❌ "우리 회중에서는…" → 외부자에게 벽이 느껴짐.
  - ✓ 교정: "저희가 속한 회중에서는…" ("저희"는 명시적으로 말하는 사람 집단만 지칭)
- ❌ "우리가 아는 대로 1914년은…" → 비증인은 이 전제를 공유하지 않음.
  - ✓ 교정: "성서가 가리키는 1914년은…" 또는 "여호와의 증인이 성서 예언을 통해 이해하는 1914년은…"

### 2-3. "우리" vs "저희" 구분

- **"우리"**: 청중 전체를 포함하는 맥락에서만.
- **"저희"**: 말하는 사람(증인 집단)만 가리킬 때. 비증인 청중 존중 표현.

## 3. 비증인 청중 직접 호명 타이밍

강연 30분 중 **2-3회** 비증인을 직접 따뜻하게 호명하는 순간이 있어야 한다:

| 타이밍 | 문구 예시 |
|--------|----------|
| 서론 중반 (약 1:30) | "오늘 처음 저희와 자리를 함께해 주신 분들도 계실 것입니다. 환영합니다." |
| 요점 2-B (약 18분) | "혹시 오늘 처음 들으시는 분이라면, 이 숫자가 현실인지 jw.org에서 직접 확인해 보실 수 있습니다." |
| 결론 (약 29분) | "오늘 이 자리에 계신 여러분 한 사람을 향해 예수 그리스도께서 이 말씀을 하십니다 — 마태 25:34 —" |

## 4. 금지 화법 체크

| 금지 | 이유 | 대체 |
|------|------|------|
| "여러분도 아시다시피 여호와의 조직은…" | 비증인은 모름. 배제감. | "여호와의 증인들이 세계적으로 수행하고 있는 활동은…" |
| "우리 형제자매들이 모두 아멘하시죠?" | 청중 반응 강요형. 공개강연은 대화형 아님. | 질문형 서술문으로: "여러분 모두 이런 경험 있지 않으십니까?" |
| "이 세상 사람들은" | 비증인을 '이 세상 사람'으로 타자화. | "우리가 사는 세상은" / "많은 사람들은" |
| "하나님의 말씀대로" | 타 교단 어감 + "여호와" 희석. | "성서의 말씀대로" / "여호와의 말씀대로" |
| "할렐루야", "아멘" (남발) | 공개강연 포맷 아님. | 삭제. |

## 5. 베트남어 번역 대비 호칭 메모

- 베트남어는 한국어보다 호칭·대명사가 더 관계 명시적 (`anh chị em` / `quý vị`).
- "형제자매 여러분" → `anh chị em` / "청중 여러분" → `quý vị` / `quý khán giả`.
- 한국어 원고에서 두 호칭을 구조적으로 섞어 두면 번역 시 대응이 쉬워진다.
