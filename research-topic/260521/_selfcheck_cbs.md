## 2. 출판 symbol 분리 의무 — 위반 0건 검증

### 2-1. 「훈」 (lfb 1102016XXX) ↔ 「예수」 책 (jy 1102014XXX) 분리

- 본 산출물의 모든 「예수」 책 챕터 인용은 docid `1102014XXX` 시리즈만 사용:
  - 1102014690, 1102014691, 1102014716, 1102014717, 1102014718, 1102014720, 1102014722
- 본 산출물에서 「훈」 (lfb) 본문 docid (1102016096 / 1102016097 / 1102016143) 는 **단순히 매핑 표에서 비교 대상으로만** 표기, cross-ref 자료로는 인용하지 않음 (지시서대로 「훈」 본문은 낭독자 담당).
- 본문 표기에서 두 책을 명확히 분리:
  - "「예수」 책 제NN장 'XX'" 형식 일관 사용
  - "「훈」 86장" / "「훈」 87장" / "「훈」 13부 소개" 는 비교 대상 표기에만 사용

### 2-2. meta.yaml 추정 정정 명시

- meta.yaml 의 추정 "jy 90장 (나사로) / jy 121장 (주의 만찬)" 은 부분 오류였음.
- 실제 wol fetch 검증 결과:
  - 나사로 부활 = jy 제**90장 + 제91장** (두 챕터 분량)
  - 주의 만찬 = jy 제**117장** (제121장 X)
- 이 정정 사실을 산출물 본문 (🟢 착수 블록 + 표) 과 종료 블록 모두에 명시하여 cbs-script 단계의 잘못된 인용을 차단.

## 4. 잠재 보완 사항 (cbs-planner 재검수 시 참고)

### 4-1. 「파」 연구용 최근호 (2018-2025) 부재 — 영향 평가

지시서 ②·③ 에서 "최근 10년 「파」 연구용" 을 명시했으나, wol 검색이 최근 timeout (서버 부하) 으로 직접 매칭되는 최근 호 확인 어려움. 다음 대안으로 충분히 보강됨:

- **86장 "예수의 눈물·사랑"**: jy 제91장 안에 권위 해설 + g70 1970845 (보조)
- **13부 "여호와의 충성"**: it-1 "부활" 항목이 더 권위 있는 표준 자료
- **87장 "겸손과 큰사람"**: jy 제118장이 직접 일화 챕터로 더 권위적

cbs-planner 가 시간 여유가 있으면 wol 재시도하여 「파」 연구용 1편 추가 인용 가능. 단 **본 산출물 단독으로도 cbs-script 작성에 충분한 권위 횡단 자료 확보** 상태.

### 4-2. 「위층 방」 통찰 항목

지시서 ③ 에 "「통찰」 (위층 방 / 다락방) — 1세기 가옥 구조" 명시되어 있으나 wol 검색에서 별도 단독 항목 hit X (지명·가옥구조 항목으로 통합 가능성). jy 제117장 본문 안에 위층 방 묘사가 포함되어 있어 대체. cbs-script 가 깊이 있는 1세기 가옥 구조 해설을 원할 경우 it-2 "예루살렘" 항목 등으로 추가 검색 가능.

### 4-3. 86장 "확장 질문" (슬픔·부활 약속에 시선 고정) 횡단 자료

이 적용은 application-builder 와 experience-collector 가 더 직접적으로 다룰 영역. 본 산출물에서는 1102015147 "죽은 사람들이 부활될 것입니다!" 가 회중 청중에게 적합한 보강 자료로 기능.
