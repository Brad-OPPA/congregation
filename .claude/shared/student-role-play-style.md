## 1.5. 재료 수집 — 3소스 통합 (가장 중요)

실연 대사는 **3개 출처를 결합**해 만든다. WOL 주차 페이지만 봐서는 절대 충분하지 않다.

### ① WOL 주차 페이지 — 과제 뼈대
`https://wol.jw.org/ko/wol/dt/r8/lp-ko/YYYY/M/D` → "집회 → 생활과 봉사 → 야외봉사에 힘쓰십시오" 블록의 해당 번호(4·5·6) 클릭.

여기서 확보:
- 과제 제목 (예: "대화 시작하기")
- 시간 (예: "2분")
- 장면 (예: "호별 방문")
- **재료 지정** (예: "「사람들을 사랑하십시오」 팜플렛 **부록 가**에 나오는 한 가지 진리")
- **학습 요점 번호** (예: "「랑제」 1과 요점 4")

### ② 재료 본문 — 대화 본론 (과제마다 재료 종류 다름)

WOL 주차 페이지가 지정한 **재료 유형** 을 확인하고 해당 페이지로 직접 이동. 유형 예시:

| 재료 유형 | 출처 | 실제 예 |
|---|---|---|
| 부록 가 진리 | 「사랑」 팜플렛 부록 가 (34개 진리 목록) | "미래·가정·하느님·기도·예수·왕국·고난·죽음·종교" 9 카테고리 |
| 성경 공부 소개 카드 | jw.org 전자 카드 | 관심 연결 후 카드 전달 흐름 |
| jw.org 카드 | jw.org 웹사이트 안내 카드 | 사이트 주요 섹션 소개 |

⚠ **카드·비디오 등 재료 상세 내용을 모를 때** — student-assignment-planner 가 `publication-cross-ref` 또는 `wol-researcher` 에이전트를 추가 호출해 wol.jw.org / jw.org 에서 해당 카드·비디오의 실제 내용·이미지·사용법을 확인해야 한다. 훈련 데이터 기억으로 카드 내용 지어내기 금지.
| 특정 비디오 | JW 방송 단편 | WOL 지정 URL 재생 |
| 특정 단락 | 「성경은 실제로 무엇을 가르치는가」·「영원한 삶」 등 | 단락 번호 명시 |
| 성구 | 직접 지정 | 본문 verbatim |

**부록 가 진리** 를 재료로 쓸 때 — 34개 중 **1개만 선택** (WOL 이 "한 가지 진리" 명시). 선택한 진리의 **연결 성구를 낭독**.

부록 가 카테고리(참고용):

- 미래 (5개: 이사 35:1,2 / 시 37:29 등)
- 가정 (4개: 엡 5:33 / 잠 1:8,9 등)
- 하느님 (4개: 시 83:18 / 시 145:18,19 등)
- 기도·예수·하느님의 왕국·고난·죽음·종교 (각 3~5개)

### ③ 조언과 (「랑제」 또는 「가르치는 기술」) 과·요점 본문 — **대본 방향 결정** ⭐⭐

**WOL 주차 과제 옆 괄호 `(「랑제」 N과 요점 M)` 가 대본 방향을 결정하는 핵심 가이드.** 과제마다 조언과가 다르고, 조언과 본문에 **"어떤 방향으로 스크립트를 짜야 하는지" 가 자세히 명시**돼 있음. WOL 에 직접 링크 걸려 있음 — 반드시 본문까지 열어 읽을 것.

조언과 본문 유형 예시:

| 조언과 | 주제 (예시) | 대본 적용 방향 |
|---|---|---|
| 「랑제」 1과 4 | 사람들에 대한 관심 — **관찰** | 집주인 옷·집·활동 관찰 멘트, 타이밍 고려 |
| 「랑제」 2과 3 | 대화 시작 — **공통 관심사** (예시) | 시사·일상 주제로 공감대 형성 후 진리 연결 |
| 「랑제」 9과 3 | 관심 자라도록 돕기 — **카드 활용** (예시) | 카드를 자연스럽게 제시·후속 연락 방법 제안 |
| 「가르치는 기술」 N과 M | 다양 | 본문 내용에 맞게 체현 |

⚠ 조언과 내용은 **훈련 기억으로 쓰지 말 것**. 반드시 WOL 에서 실제 본문을 읽고 그 **원문 핵심**을 대사 흐름으로 녹일 것. 조언과를 **설교조로 인용하거나 명시적으로 설명하지 말 것** — 대사 흐름 자체가 그 조언을 체현해야 함.

### 통합 방식 — 예시 3개

**예시 1: WOL 4번 — 대화 시작하기(2분) 호별 방문 (「랑제」 1과 요점 4)**

- 재료: 부록 가 "미래 — 환경 문제 해결·낙원 (이사 35:1,2)" 1개 선택
- 조언과 (랑제 1과 4): 관찰·타이밍
- 흐름:
  1. 집주인 관찰 후 자연스러운 인사 ("정원 가꾸시는 모습이 참 좋아 보이네요")
  2. 관심사 연결 ("요즘 환경 뉴스 보면 마음이 무거우시죠?")
  3. 부록 가 진리 제시 ("···라는 점 들어 보셨나요?")
  4. 성구 낭독 (이사 35:1,2)
  5. 후속 제안

**예시 2: WOL 5번 — 대화 시작하기(2분) 공개 증거 (「랑제」 2과 요점 3)**

- 재료: 부록 가 중 진리 1개 (예: "고난 — 하느님이 곧 모든 고난을 없애실 것 (계 21:3,4)")
- 조언과 (랑제 2과 3): 공통 관심사·공감대
- 흐름:
  1. 공공장소 상황 관찰·짧은 인사
  2. 시사·일상 공통 관심사 ("뉴스 보고 마음 무거운 분들 많으시죠")
  3. 공감·긍정 반응
  4. 부록 가 진리 자연스럽게 제시
  5. 성구 낭독 + 관심 확인

**예시 3: WOL 6번 — 관심 자라도록 돕기(2분) 공개 증거 (「랑제」 9과 요점 3)**

- 재료: 성경 공부 소개 카드 또는 jw.org 카드 (부록 가 진리 X)
- 조언과 (랑제 9과 3): 카드 활용·후속 연결
- 흐름:
  1. 이전 대화 회상·관심사 재확인
  2. 카드 자연스럽게 제시 ("이런 자료를 보실 수 있어요")
  3. 카드 내용 간단 안내
  4. 후속 연락 방법 제안

**모든 예시 공통 원칙**: 조언과가 **대사 흐름 전반을 설계**하고, 재료(진리·카드·비디오)가 **대화의 본론**이 된다. 성구는 **반드시 verbatim 낭독**.

## 2. 대사 리듬·길이

- 한 대사 **2~4문장**
- **구어체 축약형** 선호:
  - "보실까요?", "틀어드릴께요", "읽어주시겠어요", "보실까요"
  - "~이지요", "~이시죠", "~드실수 있지요"
- **감탄·호응 풍부**:
  - `네~`, `음...`, `아이고~`, `음~`, `네~~`
- 과도한 설교조·형식적 대사 **금지**
- 딱딱한 경어 대신 **친근한 경어체**

## 3. 흐름 구조 A — 연구·재방문 (apply_bible_study / apply_follow_up)

```
1. 전환 문장 — "이제 이어지는 N번 부분을 보실까요?"
2. 영상·삽화 재생 지시 (해당 시) — "여기는 동영상이 있네요. 틀어드릴께요."
3. 교재 질문을 그대로 제시 — "동영상에서 ~한 이유는 무엇입니까?"
4. 집의 답변 — 앞선 내용 요약, 구어체 자연스럽게
5. 전의 확인·긍정 — "네, 잘 기억하셨네요." / "정확하게 말씀해 주셨습니다."
6. 심화 질문 — "그럼 ~한 이유는 무엇일까요?"
7. 성구 유도 — "그래서 예수께서는 마7:21에서 이렇게 말씀하셨습니다."
8. 낭독 분담 —
   - 상대자 읽기: "여기를 어르신께서 읽어주시겠어요."
   - 학생 읽기: "여기는 제가 낭독해드릴께요."
9. 낭독 본문 verbatim (신세계역 연구용, 절 번호 인라인)
10. 적용 질문 — 성구 요점을 일상과 연결
11. 집의 적용 — 구어체 짧게 ("행동이지요. 말만 번지르하고...")
12. 전의 칭찬·정리 — "네. 맞습니다."
13. 다음 소제목/단계 전환 — "그럼 다음 소제목 7번을 보실까요."
```

## 4. 흐름 구조 B — 공개 증거 (전시대·가두·공원·주차장 등)

**공개증거 = 전시대·가두증거·공원·사업 구역·주차장** 등 **지나가는 사람** 을 만나는 장면 (S-38-KO 13항).

### 공개증거 하위 장면별 시작 멘트 패턴

**① 전시대 증거** (전시대·책꽂이 세워놓은 상태, 가까이 오거나 스쳐 지나가는 사람)

```text
전도인: 안녕하세요~ 잠깐 보고 계시네요. 관심 가는 잡지가 있으시면 무료로 보실 수 있습니다.
       이번 호 제목이 특별해서 많이들 보고 가세요. ('{잡지 제목}' 보여 주기)
집주인: (호기심) 뭐예요? / (거절) 괜찮아요, 그냥 보는 거예요.
```

**② 공원·산책로** (산책하는 사람, 벤치에 앉아 쉬는 사람)

```text
전도인: 안녕하세요~ 산책 나오셨네요. 오늘 날씨가 참 좋지요?
       요즘 이런 여유 누리기 쉽지 않잖아요.
집주인: (짧게 응답) 네, 뭐...
전도인: 실은 요즘 사람들이 이런 시간에 무엇을 생각하시는지 궁금해서 이야기 나눠보고 있어요.
       ···라는 점에 대해 생각해 보신 적 있으세요?
```

**③ 가두 증거** (길거리·번화가·건널목 신호 기다림)

```text
전도인: 안녕하세요~ 잠깐 시간 괜찮으실까요?
       요즘 많은 분들이 궁금해하시는 성경 구절을 한 가지 나누고 있는데요.
       ···라는 점 들어 보셨나요?
집주인: (바쁘다 표시 or 관심 표시)
```

**④ 주차장·버스 정류장·카페 대기** (잠시 기다리는 상태)

```text
전도인: 안녕하세요~ 기다리시는 동안 방해가 안 되신다면,
       요즘 많은 분들이 생각해 보시는 질문 하나 여쭤봐도 될까요?
       ···라는 점에 대해 어떻게 생각하세요?
```

### 공개증거 공통 흐름 (장면 무관)

```text
1. 장면 맞춤 시작 멘트 (위 ①~④ 중 해당)
2. 집주인 반응 — 호기심 / 간단 응답 / 저항 / 바쁨
3. 전도인 공감·긍정 —
   "네~ 그런 생각 드실 수 있지요."
   "선생님께서는 따듯한 분이시구나~ 생각이 들어요."
4. 집주인 저항·이야기 계속 (또는 관심 확장)
5. 관심사 연결 — 시사·가정·건강·뉴스 등 집주인 반응을 부록 가 진리로 연결
6. 부록 가 진리 1개 제시 —
   "···라는 점을 알고 계셨나요?" 또는 "···라는 점을 들어 보셨나요?"
7. 성구 낭독 (부록 가의 해당 진리 연결 성구 — NWT 연구용 verbatim)
8. 구절 보여주기 제안·후속 연결 —
   "제가 이 잡지의 한 구절을 한번 보여드리고 싶은데요. 괜찮으실까요?"
```

### 공개증거 장면 작성 시 유의 사항

- 장면은 **구역에 맞게** — 수원 연무 회중 기준 (S-38 13항 "학생은 배정된 일반적인 장면을 구역의 상황에 맞게 적용")
- 집주인의 **일상감 있는 반응** (걸음 멈추기·잠깐 쳐다보기·짜증·호기심 등)
- 짧고 강렬한 접근 — 공개증거는 **시간 제약** 이 있으므로 첫 15초 이내 핵심 질문 도달
- 거절 시 **짧게 물러나되 인사 유지** — "네, 좋은 하루 되세요" 등

## 5. 성구 인용 규칙

- 참조 먼저 말함: "마7:21", "요13:34,35"
- 본문 **신세계역 연구용 verbatim** — 한 글자도 바꾸지 말 것
- **절 번호 인라인** 표기: `34내가 여러분에게 새 계명을 줍니다. 서로 사랑하십시오. ... 35여러분 가운데 사랑이 있으면, ...`
- 인용부호 `"…"`
- 낭독자 교대 — 상대자 or 학생 (분담 명시)

## 6. 학습 요점 적용 — 체현, 표기 금지

- 원고에 `[요점 적용: 공감 기술]` 같은 **인라인 주석 금지**
- 학습 요점(「사람들을 사랑하십시오」·「가르치는 기술」 원칙) 을 대사 흐름으로 **자연스럽게 체현**
- 예:
  - **공감 기술** → "네~ 그런 생각 드실수 있지요. 저도 아주머님과 같은 생각을 할때가 많은것 같아요."
  - **인정·칭찬 기술** → "네, 잘 기억하셨네요." / "정확하게 말씀해 주셨습니다." / "기억력 너무 좋으세요."
  - **상대자 스스로 결론 유도** → 설교 대신 질문: "그럼 ~할것이라고 생각드세요?"
  - **상대자 경험 수용** → "나도 어릴 때 전쟁을 겪어봤는데" 같은 저항도 공감 후 유도

## 7. 호칭·톤

- 상대자 맞춤 호칭: **"어르신께서"**, **"아주머님"**, **"자매님"**, **"형제님"**, **"선생님"**
- 경어체 유지하되 **딱딱하지 않게**
- 친근한 어미: "~이지요", "~이시죠", "~드실수 있지요", "~더라구요", "~하시죠"

## 8. 영상·삽화 활용

WOL 이 영상·삽화를 제공하면 **반드시 대사로 언급**:

- **영상**: "여기는 동영상이 있네요. 틀어드릴께요."
- **영상 후 질문**: "동영상에서 ~의 이유는 무엇입니까?"
- **삽화**: "여기 삽화 속 ~의 모습이 보이시죠."
- 삽화 해설 → 적용 질문으로 연결

- 상대자가 **스스로 결론 내리는** 흐름
- 저항도 현실감 있게 ("내 살기도 바쁘니까.", "그치만, 나도 어릴 때...")

## 13. 사회자 칭찬·조언 블록 (원고 필수 말미, 1분 이내)

**근거**: S-38-KO 9/25 지침 **18항 전문**:

> 학생 과제가 끝날 때마다, 생활과 봉사 집회 사회자는 약 1분간 지정된 학습 요점에 근거하여 칭찬과 조언을 합니다. 집회 사회자는 학생 과제를 소개할 때 학습 요점을 알리지 않을 것입니다. 하지만 학생 과제가 끝나면, 집회 사회자는 **적절한 칭찬을 한 후 지정된 학습 요점을 언급하면서 학생이 해당 과와 관련해서 잘한 점, 좀 더 주의를 기울여야 할 점과 그 이유를 친절하게 설명해 줄 수 있습니다.** 또한 집회 사회자는 수행된 과제와 관련하여 학생이나 청중에게 유익할 만하다고 생각되는 **다른 점들**도 언급할 수 있습니다. 지정된 학습 요점이나 다른 학습 요점에 대하여 「사람들을 사랑하십시오」 팜플렛, 「가르치는 기술」 팜플렛, 「전도 학교」 책에 근거한 도움이 될 만한 조언을 **집회 후나 다른 때에 개인적으로 더** 해 줄 수 있습니다.

### 18항의 5단 구조 (원고에 반드시 이 순서로)

1. **적절한 칭찬** (먼저)
2. **지정된 학습 요점 언급**
3. **학생이 해당 과와 관련해 잘한 점**
4. **좀 더 주의를 기울여야 할 점 + 그 이유**
5. **(선택) 수행 과제 관련 학생·청중에게 유익한 다른 점**

- **적용 대상**: 학생 과제 6종 전부 (bible_reading + apply_conversation_start / follow_up / bible_study / explaining_beliefs(실연·연설))
- **시간**: 약 **1분** — 엄수 (S-38-KO 19항 시간 초과 금지)
- **담당**: 생활과 봉사 집회 사회자 (매주 한 명의 장로, 24항)
- **근거 자료**: 지정 학습 요점 — 「사람들을 사랑하십시오」 / 「가르치는 기술」 / 「전도 학교」
- **과제 소개 시 학습 요점 비공개** (18항 명시). 종료 후에만 언급.
- 추가 개인 조언은 **집회 후·다른 때** 개인적으로 (원고 범위 밖).

## 14. 과제 타입별 세부 규정 (S-38-KO 7~14항 요약)

### 공통 (6항)
- 15분 야외봉사 섹션 내 배정
- 구역에 맞게 시기적절·효과적 대화 소재 선택
- 암기 금지 — 개인적 관심·자연스럽게 말하기 (12항)
- 토의 프로는 장로 or 자격 봉사의 종 (6항)

### 7항 대화 시작하기 (apply_conversation_start)
- 남녀 가능
- 보조자: **학생과 동성 or 학생의 가족**
- 앉거나 서서 가능
- 12·13항 "내용·장면" 적용

### 8항 관심이 자라도록 돕기 (apply_follow_up)
- 남녀 가능
- 보조자: **동성만** (「사」 97/5 2면)
- 앉거나 서서
- **"지난번 대화에 이어서"** — 이전 대화의 후속으로 실연
- 12·13항 적용

### 9항 제자가 되도록 돕기 (apply_bible_study)
- 남녀 가능
- 보조자: **동성만**
- 앉거나 서서
- **이미 진행 중인 연구의 일부분** 실연
- **서론·결론 생략 가능** (학습 요점이 서론·결론에 대한 것이 아닌 한)
- 지정 자료 **낭독은 선택적** (반드시 전부 낭독할 필요 없음)

### 10항 우리의 신앙 설명하기 (apply_explaining_beliefs)
- **연설 지정**: 남자 학생만
- **실연 지정**: 남녀 가능, 보조자 동성 or 가족
- 제공된 참조 자료 내용 사용, **존중심 있고 명확하게** 주제 질문에 답
- **참조 출판물 언급 여부는 학생 결정**

### 11항 연설 (apply_talk)
- **남자 학생만** (회중 대상)
- 「사람들을 사랑하십시오」 부록 가 근거 시:
  - 부록 성구들을 **봉사에서 어떻게 사용**하는지 강조
  - 언제 성구 사용, 성구 의미, 상대방과 논리적으로 함께 생각하는 방법
- 「사랑」 팜플렛 한 과의 요점 연설 시:
  - **봉사에 적용하는 방법에 초점**
  - 해당 과 요점 1의 본 강조 or 과의 보충 성구 강조

### 12항 내용 (대화시작·관심자라만 적용)
- 별도 지시 없는 한 목표: **간단한 성경 진리 전달 + 다음번 대화 기초 놓기**
- 구역 상황 맞게 대화 소재 선택
- 봉사 도구함 출판물·동영상 사용 선택 가능
- 암기 금지 — 개인적 관심 나타내기·자연스럽게 말하기

### 13항 장면 (3종)
학생이 구역 상황에 맞게 적용:

1. **호별 방문** — 직접 만남·전화·편지, 재방문 포함
2. **비공식 증거** — 일상 대화 중 기회 포착 (직장·학교·집 주위·대중교통)
3. **공개 증거** — 전시대·사업 구역·가두·공원·주차장

### 14항 동영상·출판물 사용
- 학생이 동영상·출판물 사용 **결정 가능**
- 과제에 동영상 포함·학생이 선택 시 — **소개하고 토의**
- ⚠ **동영상 재생 금지** (학생 과제 내에서)

