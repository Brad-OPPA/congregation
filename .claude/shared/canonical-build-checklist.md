## 1. 항의 공식 질문이 출발 (정본 흐름, 2026-05-10)

각 항(블록)의 commentary 5필드는 **공식 질문에 대한 답** 의 5~6면. 별도 X, 1 흐름.

```
공식 질문 (Q)
   ↓
[answer]      Q 의 답 — 본문 기반, 사회자 정리 (verbatim 복사 X)
[depth]       Q 답을 더 깊이 — 본문 인물 × 주제 + 출판물 + 어근/지리/역사 + 각주 fetch
[key_scripture] 항에 등장한 성구의 5요소 narrative (Q 와의 연계)
[key_point]   Q 답을 어떻게 적용 (4축 + 자기점검 질문)
[real_life]   적용 카드 구체화 (구체 상황 1~2개)
[footnote]    본문에 각주·후주·"살펴보십시오" 링크 → fetch → Q 와의 관계 찾아 요점
```

## 7. 각주·후주 자동 fetch

본문 패턴 발견 시 자동 fetch:
- "(또한 ~ 살펴보십시오)"
- "각주 [a-z]"
- "후주"
- "jw.org" / "JW 라이브러리"

→ 해당 페이지 fetch → 항 요점과의 관계 → depth 안에 통합

**예시**: 18항 각주 = 게오르기 포르출랸 형제 jw.org 체험기 → 그가 통찰력을 어떻게 실천했는지 1~2 문단 정리 → 18항 depth 끝에 통합.
