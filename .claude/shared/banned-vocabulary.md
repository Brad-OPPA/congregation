## 2-ter. 추상·창작 의심 표현 (할루시네이션 가능성)

WOL WebFetch 최근 10년 매치 ≥ 1건 검증 의무. 검증 없이 사용 시 HIGH NG.

| 의심 표현 | 비고 |
|---|---|
| ...의 임재 (명사형) | 활용형 "임하시다" 는 OK. 명사형 "임재"는 wol 검증 의무 |
| ...의 현현 / ...의 강림 | 신학·창작 어휘 의심 |
| 신성한 ... / 거룩한 ... | 컨텍스트 의심 — wol 검증 |
| 성도 / 성례 / 은혜의 보좌 | 개신교 신학 어휘 |
| 1차 강림 / 2차 강림 | "강림" 은 wol 표준 X — "재림"·"오심"·"임하시다"로 |

## 3. WOL WebFetch 의무 (권장 어휘 결정)

표에 없는 의심 어휘 (예: "은혜·축복·간구·찬양 인도·새벽 기도·신학·교제") 발견 시 jw-style-checker 가 다음 절차로 결정:

1. `https://wol.jw.org/ko/wol/s/r8/lp-ko?q={의심어휘}&p=par` WebFetch
2. 매치 0건 → 그 어휘는 wol 표준 X → **권장 어휘 결정 의무** (전후 문맥에서 신세계역 어휘 선택)
3. 매치 풍부함 → 그 어휘는 wol 표준 O → 유지
4. 산출 보고서에 **"WebFetch wol.jw.org 결과: [어휘 매핑]"** 섹션 의무 — 없으면 자체 검수 FAIL

## 4. 컨텍스트 예외 (validators.py `scan_forbidden_words` 자동 적용)
